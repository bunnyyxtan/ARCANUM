# Community

Arcanum is for builders on Arc: DAO treasury operators, DeFi protocol engineers, DePIN operators, and on-chain agent teams.

- GitHub: `https://github.com/bunnyyxtan/ARCANUM`
- Discussions: coming with the public repo
- Discord: coming with launch
- Twitter: `@arcanum_protocol`

## Contributor guide

Start with a small issue, include tests, and keep protocol commitments crisp. Docs contributions are welcome, especially real operator recipes from Arc builders.
