# Escalations

Escalations turn high-risk transfers into pending on-chain requests. Required signers approve or reject. Expired escalations settle to reject by default.

```ts
const unwatch = arc.onEscalationResolved("0xabc123" as `0x${string}`, (event) => {
  console.log(event.status)
})
```

Council review is explicit authorization, signed by the operators chosen by the wallet owner.
